# Tower Builder

Tower Builder is a physics-based mobile game developed in Unity 3D. The game challenges players to strategically stack blocks and structures to build the tallest and most stable tower possible while dealing with dynamic obstacles such as wind and birds.

## Overview

Tower Builder is designed as a casual yet skill-based mobile game for Android. Players must carefully place different building blocks, maintain balance, and prevent the tower from collapsing. As the levels progress, the game introduces new challenges that increase difficulty and encourage strategic thinking.

## Features

- Physics-based tower stacking system
- Level-based progression with increasing difficulty
- Dynamic obstacles such as wind and birds
- Multiple background themes including city, forest, ocean, and space
- Score system based on tower height, stability, and performance
- Simple and engaging gameplay suitable for short play sessions
- Designed for Android mobile devices

## Gameplay

Players stack buildings and blocks to create a stable tower. Each level presents different objectives and challenges. Early levels focus on basic stacking mechanics, while later levels introduce additional blocks, environmental effects, and obstacles that test the player’s control and strategy.

The main objective is to build the tallest possible tower without letting it collapse.

# Tower Builder - Physics-Based Stacking Game

## Overview
Tower Builder is a **physics-based tower-building game** where players stack blocks to build the tallest and most stable structure possible. The game incorporates **realistic physics, dynamic obstacles, and level-based progression**, making it both engaging and challenging.

## Features
### 1. **Physics System & Stacking Mechanics**
- Drag & drop blocks to place them on the tower
- Realistic gravity, friction, and collision interactions
- Block snapping to improve placement accuracy

### 2. **Dynamic Obstacles**
- **Wind Effect:** Random wind forces affect the blocks
- **Birds:** Moving birds collide with and knock over blocks

### 3. **Level-Based Progression & Scoring**
- Each level introduces new challenges
- Players must reach a specific tower height to progress
- Score is based on stability and height of the tower

### 4. **User Interface (UI)**
- **Main Menu:** Play, Settings, and Exit options
- **In-Game UI:** Displays score, level, and restart button
- **Game Over Screen:** Retry and Home options

### 5. **Background Themes**
- Switchable environments: City, Forest, Ocean, and Space
- Low-poly assets for optimized performance

## How to Play
1. Drag and drop blocks to build a stable tower.
2. Avoid obstacles like wind and birds.
3. Reach the required height to complete each level.
4. Progress through increasing difficulties and earn points.

## Technologies Used
- **Game Engine:** Unity 3D (2022.3.13 LTS)
- **Programming Language:** C#
- **Physics Engine:** Unity's built-in physics system

## Installation & Setup
1. Clone the repository: `git clone https://github.com/your-repo/tower-builder.git`
2. Open the project in **Unity 2022.3.13 LTS**
3. Build and play the game!
